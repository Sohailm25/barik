# Utils - Utility Functions and Extensions

Helper functions, extensions, and utility classes.

## Files

- **ExperimentalConfigurationModifier.swift** - SwiftUI view modifier for experimental features
- **ImageCache.swift** - Caching system for downloaded images (album artwork, etc.)
- **VersionChecker.swift** - Checks for app updates from GitHub releases
- **View+Extensions.swift** - SwiftUI View extensions for common functionality
- **VisualEffectBlur.swift** - NSVisualEffectView wrapper for macOS blur effects
