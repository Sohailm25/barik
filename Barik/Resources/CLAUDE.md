# Resources - Application Assets

Contains all visual assets and localization files.

## Files

- **Localizable.xcstrings** - Localized strings for internationalization

## Folders

- **Assets.xcassets/** - Xcode asset catalog containing:
  - **AccentColor.colorset/** - Accent color definitions
  - **AppIcon.appiconset/** - Application icons in various sizes
  - **Colors/** - Custom color sets for UI theming (Active, Background, Foreground, etc.)
  - **Icons/** - Custom icon assets (Bluetooth symbol)
  - **MenuBar.imageset/** - Menu bar icon
