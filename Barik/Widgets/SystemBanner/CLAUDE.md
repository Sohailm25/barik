# SystemBanner - System Notification Banners

Displays update notifications and changelog information.

## Files

- **SystemBannerWidget.swift** - Container for system-wide banner notifications
- **UpdateBannerWidget.swift** - Shows available app updates with download link
- **ChangelogBannerWidget.swift** - Displays changelog after updates
- **ChangelogPopup.swift** - Detailed changelog viewer with markdown support
- **MarkdownTheme.swift** - Custom markdown rendering theme for changelogs
