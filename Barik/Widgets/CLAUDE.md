# Widgets - Menu Bar Widget System

All available menu bar widgets and their implementations.

## Folders

- **Battery/** - [Battery status widget](Battery/CLAUDE.md) - Shows battery level and charging status
- **Bluetooth/** - [Bluetooth widget](Bluetooth/CLAUDE.md) - Displays Bluetooth connection status
- **Network/** - [Network widget](Network/CLAUDE.md) - Shows network connectivity status
- **NowPlaying/** - [Now Playing widget](NowPlaying/CLAUDE.md) - Media playback controls and info
- **OutputAudio/** - [Audio output widget](OutputAudio/CLAUDE.md) - Audio device selection
- **Spaces/** - [Spaces widget](Spaces/CLAUDE.md) - Workspace/space management for tiling WMs
- **SystemBanner/** - [System banners](SystemBanner/CLAUDE.md) - Update notifications and changelog
- **Time+Calendar/** - [Time and calendar widget](Time+Calendar/CLAUDE.md) - Clock and calendar events
