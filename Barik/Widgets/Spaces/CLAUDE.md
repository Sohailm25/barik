# Spaces - Workspace Management Widget

Displays current workspace/space for tiling window managers.

## Files

- **SpacesWidget.swift** - Current space indicator with window preview
- **SpacesViewModel.swift** - Manages space data and window information
- **SpacesModels.swift** - Data models for spaces and windows
- **MenuBarViewModel.swift** - Integration with menu bar for space switching

## Folders

- **Aerospace/** - [AeroSpace WM integration](Aerospace/CLAUDE.md) - Support for AeroSpace tiling WM
- **Yabai/** - [Yabai WM integration](Yabai/CLAUDE.md) - Support for Yabai tiling WM
- **Views/** - [Space UI components](Views/CLAUDE.md) - Visual components for spaces
