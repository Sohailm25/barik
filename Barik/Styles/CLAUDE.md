# Styles - UI Styling Components

Custom SwiftUI button and progress styles.

## Files

- **DefaultButtonStyle.swift** - Standard button style with hover and press states
- **DragButtonStyle.swift** - Button style for draggable elements
- **GradientStyles.swift** - Gradient definitions and styling utilities
- **StaticButtonStyle.swift** - Non-interactive button appearance
- **TrackProgressStyle.swift** - Custom progress bar style for media playback
- **TransparentButtonStyle.swift** - Transparent button with subtle interactions
